Politique de confidentialité

1. Données collectées

  • Bot Discord : Données collectées uniquement pour le bon fonctionnement des commandes (ID utilisateur, ID serveur, messages si nécessaire, etc.).
  • Site web : Aucune donnée personnelle n’est collectée automatiquement sur ce site.
  • Paiement : Les paiements sont traités via PayPal. Aucune donnée bancaire n’est stockée ou collectée par Project : Delta.

2. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Faire fonctionner les commandes et fonctionnalités du bot
  • Gérer les accès à la version premium
  • Garantir la sécurité et la modération si les options de sécurité sont activées

3. Stockage & sécurité

  • Les données sont stockées dans une base sécurisée (MongoDB).
  • Aucun partage avec des tiers sans consentement explicite.
  • Les accès sont strictement limités à l’équipe du projet.

4. Vos droits

Conformément au RGPD, vous avez les droits suivants :

  • Droit d’accès à vos données
  • Droit de modification ou de suppression
  • Droit d’opposition à l’utilisation

Pour exercer vos droits, contactez-nous via le serveur support.

5. Cookies

Ce site n’utilise aucun cookie ou traceur tiers.

6. Contact

Pour toute question relative à la confidentialité : Serveur Discord officiel.
